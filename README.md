# AngularJS Project Template

## Quick Start

Install Node.js:

```sh
$ brew install nodejs
```

and then:

```sh
$ git clone git@github.com:liambeeton/angularjs-kickoff.git
$ cd angularjs-kickoff
$ sudo npm -g install grunt-cli karma bower
$ npm install
$ bower install
$ cp -r bower_components ./vendor
$ npm start
```
